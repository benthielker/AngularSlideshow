Angular Slideshow Examples

Horizontal Linear Slideshow with Auto-advance

This is an example of the most common type of slideshow. Slides are arranged horizontally, automatically advance, and loop back around to be beginning when when advancing beyond the last slide. 


<div id='example1'
     loopDimensions='0'
     transitionDuration='750'
     advanceDelay='4000'
     advanceChange='1'
     style='float:right; 
          width:300px; height:150px;
          margin:1em 0 2em 3em'
     a-slideshow >
</div>

Horizontal Linear Slideshow with Manual Navigation

This example is similar to the previous one, except that it does not auto-advance, and does not loop. Lack of auto-advance configuration results in automatic generation of navigation buttons needed to access the slides present. In this case only Left and Right buttons are needed, but Up and Down buttons may also be created if applicable.

This show has also been configured to display page numbers via the show_page_numbers attribute. This only needs to be provided a value which equates to true in a boolean comparison. The page numbers indicate the current page, and when clicked transition directly to the slide their index attribute refers to. 


<div id='example2'
     loopDimensions='0'
     transitionDuration='500'
     showPageNumbers='1'
     style='float:right; 
          width:300px; height:150px;
          margin:1em 0 2em 3em'
     a-slideshow >
</div>

Vertical Linear Slideshow with Manual Navigation

This example demonstrates vertically arranged slides, does not auto-advance, and loops around the vertical axis. You will notice that the navigation controls are now positioned on the top and bottom instead of the left and right.

In this example the navigation labels have been overridden via the directionLabels attribute, which must be provided in valid JSON format. 


<div id='example3'
     loopDimensions='1'
     transitionDuration='500'
     directionLabels='{"up":"Prev","down":"Next"}'
     style='float:right; 
          width:300px; height:150px;
          margin:1em 0 2em 3em'
     a-slideshow >
</div>

2d Grid with Looping

This example demonstrates the two-dimensional abilities of the Angular Slideshow. Slides are arranged on a 3x3 grid which loops around both axes. This requires providing an array of dimension indexes to the loopDimensions attribute. (This helps explain the '0' and '1' values specified for the previous slideshows. All slides have a 2-dimensional index position, and the loop dimensions configuration indicates which axis to loop.)

This example also overrides all potential direction labels. 


<div id='example4'
     transitionDuration='750'
     loopDimensions='[0,1]'
     directionLabels='{"up":"North","right":"East","down":"South","left":"West"}'
     style='float:right;
          width:300px; height:150px;
          margin:1em 0 2em 3em'
     a-slideshow >
</div>

For a more elaborate example of what is possible with the 2d slideshow, check out the Gem Hunter example.
